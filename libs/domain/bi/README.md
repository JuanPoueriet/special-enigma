# BI Domain

## Propósito
El dominio de Business Intelligence (BI) se encarga de la generación de informes, análisis de datos y provisión de métricas para los dashboards de la plataforma.

## Perfil
**Perfil:** `service-domain`

## Responsabilidades
- Generación de informes analíticos.
- Agregación de datos de ventas, facturación y gastos.
- Provisión de estadísticas para dashboards en tiempo real.
- Integración con otros dominios para la recolección de datos (Sales, Billing, Expenses).
